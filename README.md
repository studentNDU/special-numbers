Репозиторій містить приклади реалізації візуалізацій чисел спеціального типу засобами мови Python.  
Матеріали призначені для:
- супроводу наукової публікації;
- використання в освітньому процесі підготовки вчителів математики;
- відтворення та модифікації дослідницьких завдань з теорії чисел.
